# ⚡ Calculadora de Voltaje en Java

## Descripción

Aplicación desarrollada en Java para realizar cálculos eléctricos mediante una interfaz gráfica.

El proyecto fue desarrollado como parte de mi formación en programación y permite calcular el voltaje utilizando la Ley de Ohm, aplicando validación de datos y una interfaz intuitiva para el usuario.

## Tecnologías utilizadas

- Java
- Java Swing
- Eclipse IDE
- Programación Orientada a Objetos

## Funcionalidades

- Cálculo de voltaje.
- Validación de datos ingresados.
- Interfaz gráfica amigable.
- Manejo de excepciones.

## Próximas mejoras

- Agregar cálculo de corriente.
- Agregar cálculo de resistencia.
- Mejorar la interfaz gráfica.
